Reservation system

Reservation system for registering children for enrollment into the first grade. The project is written in PHP language with integration to a MySQL database, securely protected with hashed passwords, and safeguarded against SQL injection attacks. It includes all the basic functionalities such as confirmation emails, editing, searching, deleting, and recording. The project's source code is traditionally accessible on GitHub at this link. The project is currently in the BETA phase and will undergo practical testing and further adjustments.

Hashed passwords are a classic security element used for secure login to admin access.
