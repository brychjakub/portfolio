Converting Manual Work into a Digital One

Program for calculating scores to determine the ideal future profession of students.

The source code is again on GitHub.

Problem: The long and strenuous, repetitive work of the school psychologist, which involves transcribing 240 manually circled questions into numbers and then summing them up.

Final solution: An online questionnaire done with vanila JavaScript where students generate a .txt file by clicking on the answers, which the psychologist can easily use in subsequent work. This saves an hour of work per student.

Original solution: Originally converted into a clickable Excel spreadsheet for compatibility with school PCs.

Final Solution

JavaScript - a web application accessible from mobile, tablet, and PC is the simplest option.




Original Solution

Excel - software that works on all school devices and is familiar to both students and teachers.
