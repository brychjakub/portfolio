Backup of grades into organized tables.

Problem: The need for manual downloading/backup of grades for every class, every month.

Solution: Using the Playwright library to obtain data after logging into the locally run application. This way, any colleague can also use the program.

The source code is again on GitHub.

At the beginning, the program will prompt you for login credentials, with the password hidden by asterisks.



After about half a minute, the program is finished and creates a folder named after the user.



It then copies the requested .xlsx files with the names of the classes to which the grades belong into that folder.



In the spreadsheets, there are names, averages, grades, and the subjects for which the grades are assigned.
